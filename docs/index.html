Study Palette

BDC Meta-Analysis Study Builder & Query Tool

Live Demo

About

Study Palette is a unified platform for cross-study data discovery and meta-analysis study building within the NHLBI BioData Catalyst ecosystem. It replaces fragmented search interfaces with a semantic, modular platform that enables researchers to discover data, explore variables, build studies, and transition to analysis.

Architecture

Front End

ReactJS with semantic search, query builder, and interactive visualizations.

Modular APIs

FastAPI services for search, query, analysis, and workflow orchestration.

Metadata Index

LinkML-based "source of truth" enabling consistent cross-study search at variable and participant levels.

Integrations

Monarch ontologies, DMC data ingestion, BDC analytic widgets, and foundational BDC services.
